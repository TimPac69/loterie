🚀 Mise en ligne en 10 minutes

Suivez ces étapes pour héberger votre loterie gratuitement sur GitHub Pages. Vos invités pourront scanner le QR code depuis n'importe quel téléphone.

Partie 1 — Créer un compte GitHub

1
Allez sur github.com et créez un compte gratuit

Cliquez "Sign up", entrez votre email, choisissez un nom d'utilisateur et un mot de passe. Confirmez votre email.

Partie 2 — Créer un dépôt (dossier en ligne)

2
Créez un nouveau dépôt

Une fois connecté, cliquez sur le + en haut à droite → "New repository".

3
Nommez le dépôt exactement : loterie

Laissez tout le reste par défaut. Cochez "Public". Cochez "Add a README file". Puis cliquez "Create repository".

Partie 3 — Déposer vos fichiers

4
Cliquez "Add file" → "Upload files"

Vous êtes maintenant dans votre dépôt. Cliquez sur ce bouton en haut à droite de la liste de fichiers.

5
Glissez-déposez vos 3 fichiers HTML

Faites glisser loterie-admin.html, loterie-participant.html et GUIDE.html dans la zone prévue.

6
Cliquez "Commit changes" (bouton vert en bas)

Laissez le message par défaut. Vos fichiers sont maintenant en ligne.

Partie 4 — Activer GitHub Pages

7
Allez dans Settings → Pages

Dans votre dépôt, cliquez sur l'onglet "Settings" (icône ⚙ en haut), puis dans le menu gauche cliquez "Pages".

8
Activez Pages depuis la branche main

Sous "Branch", choisissez main dans le menu déroulant, laissez / (root), puis cliquez "Save".

9
Attendez ~1 minute puis rafraîchissez la page

Un bandeau vert apparaît avec votre URL publique :

https://votre-pseudo.github.io/loterie/loterie-admin.html

Partie 5 — Configurer Firebase avec la nouvelle URL

10
Autorisez votre domaine dans Firebase

Firebase Console → votre projet → Authentication → "Sign-in method" → onglet "Authorized domains" → "Add domain" → collez votre-pseudo.github.io.

11
Ouvrez la page admin en ligne et reconfigurez Firebase

Allez sur https://votre-pseudo.github.io/loterie/loterie-admin.html, entrez vos identifiants Firebase (les mêmes qu'avant) + l'URL de la base, cliquez "Connecter".

12
Regénérez le QR code

Cliquez "Générer QR Code" — il pointera désormais vers votre URL publique. Imprimez-le et vos invités peuvent scanner !

✅ Résumé de vos URLs finales

Page admin : https://votre-pseudo.github.io/loterie/loterie-admin.html
Page invité : https://votre-pseudo.github.io/loterie/loterie-participant.html

⚠️ Remplacez votre-pseudo par votre nom d'utilisateur GitHub.

En cas de blocage à une étape, décrivez le problème — je vous aide pas à pas.
